Development Experience:

Developing the Ampersand website was a collaborative effort that focused on creating a seamless user experience. The project involved:

Planning & Strategy: Defining the website's objectives and target audience.

Design: Creating wireframes and prototypes to visualize the user interface.

Development: Coding the website using modern web technologies.

Testing: Conducting usability testing to ensure functionality across devices.

Launch: Deploying the website and monitoring performance.

Tech Stack:

Frontend:

HTML5: For structuring content.

CSS3: For styling and layout.

JavaScript: For interactivity and dynamic content.

React.js: For building user interfaces.

Backend:

Node.js: For server-side scripting.

Express.js: For building web applications and APIs.

Database:

MongoDB: For storing data in a flexible, JSON-like format.

DevOps:

Docker: For containerization and consistent development environments.

AWS: For hosting and cloud services.

Tools & Libraries:

Webpack: For module bundling.

Babel: For JavaScript transpiling.

Jest: For testing JavaScript code.

Sass: For advanced CSS styling.

This tech stack was chosen to ensure scalability, maintainability, and performance.
